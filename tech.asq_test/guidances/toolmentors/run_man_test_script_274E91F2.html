Основное описание

Overview

When you run a manual test script, you use the IBM® Rational® Manual Tester Execution viewer to go through the statements one at a time. In this way, you are able to check off steps as you complete them, and validate verification points and reporting points. (On the difference between verification points and reporting points, see Tool Mentor: When to Use Verification Points versus Reporting Points). This tool mentor describes how to prepare and run test scripts in Manual Tester.)

Tool Steps

To run a script using Rational Manual Tester, perform the following steps:

  1. Prepare the Test Environment
  2. Run the Script

1. Prepare the Environment

  1. To prepare for running a test script, open the application under test.
  2. Run Rational Manual Tester.
  3. Open the test script you want to run.

Remember to pay close attention to places in your test script that involve text insertion or that involve image files. (  For more information on entering text or using image files in your script, see the topics About Entering and Verifying Text in an Application and Inserting an Image File in the Rational Manual Tester online Help.) You may wish to focus on these areas in your script when you are running the script from the Execution viewer.

2. Run the Script

  1. With the test script open, select Run > Run Script.
  2. Rational Manual Tester displays the Execution viewer with the first statement of the script highlighted. You can move the Execution viewer over the application under test. Since the Execution viewer is transparent, both the script you are running and the application under test are visible.
  3. In the Execution viewer, review each step as you progress through the script. As you progress through the script, Manual Tester places a small check mark to the left of the first step and highlights the second statement.
