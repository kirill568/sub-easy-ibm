Основное описание

Overview

When creating test scripts, you will find that some steps are prime candidates for reuse. The ability to simply drag and drop step content from one test script to another that you are working on can dramatically improve the efficiency of the test script development process.

One of the key features of IBM® Rational® Manual Tester is the ability to easily reuse content from a test script. When you are executing test scripts, you can build test scripts more efficiently by reusing test script content from previous scripts. In Rational Manual Tester, you can take statements from a given test script -- that you know will be reused in other test scripts -- and store them for reuse.

Tool Steps

To reuse content form a test script, perform the following steps:

  1. Open the script containing the content that you want to reuse
  2. In the Outline view, select the given statement, statements, or group of statements that you want to reuse
  3. Drag your selection to the Reuse view
  4. Create a new script (see the Tool Mentor: Create Test Scripts Using Rational Manual Tester)
  5. In the Reuse view, select the step you want to reuse and drag it to the Outline view. The Outline view and Text Editor are automatically updated
  6. Save your new script

For more information on reusing test script content, see the topic Saving and Reusing Test Statements in the Rational Manual Tester online Help.
