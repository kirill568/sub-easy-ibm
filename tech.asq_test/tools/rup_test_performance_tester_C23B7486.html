Основное описание
IBM® Rational® Performance Tester is for testing Web applications. It divides performance testing into five related tasks: test creation, test editing, workload emulation with schedules, schedule execution, and evaluation of results. With Rational Performance Tester, you can create tests, run schedules, represent workloads, and evaluate the results of a test run, conveniently and from one environment.
