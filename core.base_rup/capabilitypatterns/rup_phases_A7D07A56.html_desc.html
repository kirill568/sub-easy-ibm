Примечания по использованию

Для создания процесса поставки со всеми этапами RUP выполните следующие действия:

  1. Создайте новый процесс поставки
  2. Скопируйте (а не расширьте) все этапы из этого шаблона возможностей в новый процесс поставки.   Можно выбрать и скопировать все этапы одновременно. Примечание:  это самый быстрый способ создания структуры этапов RUP вместе с документацией и каркасами диаграмм операций.   После создания копии шаблона по мере определения итераций надо будет удалить отдельные элементы из этого шаблона.)
  3. Создайте итерации для всех этапов.   Выполните следующую процедуру для каждого этапа: 
    • Скопируйте (или расширьте) шаблон возможностей итерации Classic RUP в соответствующий этап процесса поставки.   Обязательно скопируйте весь шаблон возможностей, а не только операции из него. Это нужно для того, чтобы вместе с операциями были скопированы диаграммы.   Если ивы планируете изменить итерацию Standard RUP в своем процессе, это будет гораздо удобнее проделать с копией шаблона итерации, а не с расширенным шаблоном. 
    • Удалите итерацию, скопированную из этого шаблона возможностей.  В только что скопированном шаблоне итерации этапа есть свой заголовок итерации.
    • Переименуйте итерацию, если вы планируете включить в этап несколько итераций.
    • Повторите эти шаги для создания нужного количества итераций для этапа.
    • Переместите веху в конец этапа.
  4. Обновите диаграммы операций для следующих объектов:
    • Процесс поставки
    • Все этапы: начальный, уточнение, построение и внедрение
    • Все итерации всех этапов.
    • Все расширенные шаблоны возможностей, измененные в вашем процессе поставки.
    • Все операции, которые были созданы в процессе поставки с нуля и у которых при этом есть дочерние операции.
    • Примечание: если после публикации диаграммы операций не будут показаны или будут показаны неправильно, это значит, что они либо они были созданы, либо нуждаются в обновлении.    Для устранения этой ошибки откройте процесс поставки в Rational Method Composer. Если вы скопировали другой шаблон возможностей или создали собственную операцию с дочерними операциями, диаграмму операции нужно будет создать вручную.  Если вы расширили другой шаблон возможностей, должно быть достаточно обновить диаграмму из базового шаблона возможностей следующим способом:
      • откройте диаграмму операции
      • откройте контекстное меню диаграммы и выберите опцию "обновить из базы"
      • сохраните и закройте диаграмму 
  6. В результате процесс поставки будет наполнен данными из процесса Classic RUP, в котором предусмотрено достаточное количество итераций для вашего проекта.  После этого можно настроить процесс поставки с учетом особенностей вашего проекта.   Обязательно проверьте документацию, полученную из шаблона возможностей процесса поставки, и внесите в нее необходимые изменения.
