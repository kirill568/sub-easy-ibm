Инструмент: Rational Application Developer
Rational Application Developer представляет собой интегрированную среду разработки для построения, тестирования и развертывания приложений J2EE , WebServices и Portal. Предоставляется широкий набор инструментов для поддержки ускоренной разработки приложения и увеличения эффективности работы разработчиков.
Взаимосвязи
Руководства по инструментам
  • Разработка пользовательских Web-интерфейсов с помощью JSF и SDO
  • Компоновка модулей и приложений J2EE с помощью RAD 6.0
  • Развертывание Web-приложения на WebSphere Application Server
Основное описание
Приложение IBM® Rational® Application Developer представляет собой сложную интегрированную среду разработки с полной поддержкой программной модели J2EE, включая Web, Javatm, Web-службы и разработку EJB, что ускоряет разработку приложений. Включая встроенную разработку Portal, визуальное редактирование UML, анализ исходного кода и автоматизированные инструменты тестирования и развертывания, Rational Application Developer содержит все необходимое для того, чтобы обеспечить эффективную работу разработчиков и гарантировать правильную разработку, масштабируемость и готовность исходного кода для производства. Встроенные инструменты управления версиями и инструменты коллективной работы обеспечивают для разработчиков, работающих над сложными проектами или в составе крупных коллективов, возможность координации версий и защиты коллективных ресурсов.
