Инструмент: Rational Software Architect
Rational Software Architect представляет собой интегрированную среду разработки для моделирования, построения, тестирования и развертывания приложений.
Основное описание

Приложение IBM® Rational® Software Architect объединяет все аспекты проектирования и разработки программного обеспечения в одном инструменте. Он поддерживает понимание, проектирование, управление и развитие решений и служебных функций предприятия. Этот продукт включает все функции J2EE, Web и Web-служб, содержащиеся в IBM® Rational® Application Developer для WebSphere Software. Rational Software Architect построен на вершине открытой и расширяемой платформы Eclipse, усиливающей несколько открытых промышленных стандартов. Это позволяет пользователям создавать приложения, оптимизированные для промежуточного ПО IBM, а также разработанные с использованием технологии промежуточного ПО от других производителей.
