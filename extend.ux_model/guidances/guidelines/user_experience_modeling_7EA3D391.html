Рекомендация: User Experience Modeling
This guideline describes how to produce a model of the user-experience.

Основное описание

During Task: Design the User Interface, a UML model of the user interface may be developed. That model is called the  User-Experience Model.

Modeling the user experience involves the following steps, which are focused on the Use Cases being developed for the current iteration:

  • Identifying the  User-Experience Elements that provide the user interface for the Use Cases. For more information, see  Guideline: User-Experience Element.
  • Developing  User-Experience Storyboards that describe how the User-Experience Elements interact to perform the Use Cases. For more information, see  Guideline: User-Experience Storyboard.
  • Developing the  User-Experience Navigation Map that describes the navigation paths between the User-Experience Elements that support their interactions. For more information, see  Guideline: User-Experience Navigation Map.

Each of these artifacts are part of the  User-Experience Model. For more information on the User-Experience Model, see  Guideline: User-Experience Model.
