Content

The RUP for User-Experience Modeling Plug-in describes a process for transforming the requirements into a model of the user experience. It includes step-by-step instructions for developing the User-Experience Model, as well as guidance on integrating the User-Experience Model with other system models (e.g., Use-Case Model, Design Model, etc.).

The RUP for User-Experience Modeling Plug-in also provides tool mentors that describe how to model the User-Experience Model using Rational tools.

To get started using the asset-based development plug-in, see Roadmap: Modeling the User Experience.
