Основное описание

Overview

SoDA automates the generation of a User-Experience Storyboard Report so that it is created quickly and accurately. You can generate a User-Experience Storyboard Report with either the Microsoft® Word® or Adobe® FrameMaker® version of SoDA.

The following is some related Rational Unified Process information:

  •  Report: User-Experience Storyboard
  •  Work Product: User-Experience Storyboard

This tool mentor is applicable when running Windows 2000, NT 4.0, or Windows XP, Solaris, or HP-UX.

To create a User-Experience Storyboard Report using Rational SoDA, use the procedure for your version of the product:

  • Use Rational SoDA/Word to generate a User-Experience Storyboard Report
  • Use Rational SoDA/FrameMaker to generate a User-Experience Storyboard Report

Use Rational SoDA/Word to generate a User-Experience Storyboard Report

  1. In Rational Rose, select the User-Experience Storyboard you want to document.
  2. Click Report > SoDA Report.
  3. When the list of available reports appears in SoDA, select Rational Unified Process User-Experience Storyboard Report. If this report is not on the list, cancel and make sure you have a User-Experience Storyboard selected.
  4. Click OK to generate the report.

Use Rational SoDA/FrameMaker to generate a User-Experience Storyboard Report

  1. From the FrameMaker button bar, click New. Double-click SoDA, then double-click RoseDomain and choose the RUPUserExperienceStoryboard.fm template.
  2. Edit the Connector and enter the name of the model file, the parent package, and the name of the User-Experience Storyboard.
  3. Click File > Save As to save the template to a personal or project directory. You may want to change the name of the template to reflect the name of the User-Experience Storyboard; for example, MaintainLoansReport.fm.
  4. Click SoDA > Generate Document.
  5. Review the generated document.

The next time you want to generate this same document, open the document and click SoDA > Generate Document.
